# 🎓 Cloud Based Student Portal Management System

## 🚀 Project Overview

The **Cloud Based Student Portal Management System** is a modern Full Stack Web Application developed to simplify and digitalize student management operations. The application provides secure student registration, login authentication, database management, and cloud-based deployment using AWS EC2 infrastructure.

This project demonstrates real-world implementation of Full Stack Web Development, REST API integration, cloud deployment, database connectivity, and modern UI development.

# ⚙️ Installation & Setup Guide

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/vprijwal-bit/student-portal.git
```

---

## 2️⃣ Navigate to Project Directory

```bash
cd student-portal
```

---

## 3️⃣ Install Frontend Dependencies

```bash
npm install
```

---

## 4️⃣ Install Backend Dependencies

```bash
cd backend
npm install
```

---

# 🗄️ Database Configuration

## Create Database

```sql
CREATE DATABASE student_portal;
```

---

## Create Students Table

```sql
CREATE TABLE students (

  id INT AUTO_INCREMENT PRIMARY KEY,
  name VARCHAR(100),
  email VARCHAR(100),
  department VARCHAR(100),
  password VARCHAR(100)

);
```

---

# ▶️ Run Backend Server

```bash
cd backend
node server.js
```

---

# ▶️ Run Frontend Application

```bash
npm run dev
```
